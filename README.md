# 📝 ReadmeGenius

> Generate beautiful GitHub READMEs instantly — no more blank page syndrome.

## ✨ Features
- ⚡ Live preview as you type
- 🛠️ Pick your tech stack with one click
- 📋 Copy markdown or download as .md
- 🎨 Clean dark terminal-inspired UI
- 🔧 Toggle sections on/off
- 📊 Auto-generates shields/badges

## 🛠️ Built With
- HTML, CSS, JavaScript
